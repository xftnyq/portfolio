How I Started My Journey

I'm sharing how my programming journey began and the valuable lessons I've learned along the way. It all started one summer break when I stumbled upon an online coding tutorial out of sheer curiosity. With no background in programming, I was simply eager to understand how websites functioned behind the scenes. Writing my first 'Hello, World!' in HTML felt like discovering something magical. That initial interest soon led me to explore CSS, JavaScript, and later VB.NET in my coursework. Despite the many bugs and moments of frustration, each bit of progress brought a sense of achievement. Today, programming is more than just a skill—it's a passion I’m constantly developing.

Visual Studio Code

Visual Studio Code is my preferred development environment, thanks to its clean, lightweight interface and robust features. Tools like IntelliSense, the integrated terminal, a wide range of extensions, and seamless Git support have significantly enhanced my productivity. Whether I’m working with VB.NET, fine-tuning HTML/CSS, or diving into Python, VS Code helps keep my workflow streamlined and organized. Its customization capabilities and vast extension library allow me to personalize the workspace, making development not only efficient but also enjoyable.
